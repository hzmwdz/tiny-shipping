# TinyShipping

## Install

```bash
composer require hzmwdz/tiny-shipping
```

## Publish

```bash
php artisan vendor:publish --provider="Hzmwdz\TinyShipping\TinyShippingServiceProvider" --tag="migrations"
php artisan vendor:publish --provider="Hzmwdz\TinyShipping\TinyShippingServiceProvider" --tag="translations"
```
